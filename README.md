# Paper-Rock-Scissors Game
## Live Link https://sujan2017.github.io/Paper-Rock-Scissor-Game/
A simple **Paper-Rock-Scissors game** built using **HTML, CSS, and JavaScript**.  
This mini project demonstrates **DOM manipulation, game logic, and interactive user interface** with JavaScript.

##  Features

- Player vs Computer gameplay
- Player selects **Paper, Rock, or Scissors**
- Computer randomly chooses a move
- Displays winner of each round
- Keeps track of scores for both user and computer
- Highlights winning and losing choices
- Option to **Reset Game** or **Start New Game**
- Detects draw rounds

##  How to Play

1. Open `index.html` in a web browser.
2. Click on **Paper**, **Rock**, or **Scissors** to make your move.
3. The computer will randomly select its choice.
4. The winner of the round will be displayed along with updated scores.
5. Click **Reset Game** to restart scores, or **New Game** to reset the game and clear messages.
